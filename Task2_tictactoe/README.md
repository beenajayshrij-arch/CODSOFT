# AI Tic-Tac-Toe

## Description
This project is an AI-powered Tic-Tac-Toe game developed using Python. The AI opponent uses the Minimax algorithm to make optimal moves, making it difficult to beat.

## Features
- Human vs AI gameplay
- Intelligent AI using Minimax algorithm
- User-friendly command-line interface
- Detects win, loss, and draw conditions

## Technologies Used
- Python

## How to Run

1. Open the terminal.
2. Navigate to the project folder.
3. Run:

python tic_tac_toe.py

## Project Outcome
This project demonstrates the implementation of game logic and Artificial Intelligence using the Minimax algorithm.
