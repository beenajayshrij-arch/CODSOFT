# Image Captioning

## Description
This project generates captions for images using a pre-trained Transformer model from the Hugging Face library. It predicts meaningful descriptions for input images using deep learning techniques.

## Features
- Automatic image caption generation
- Supports JPG image files
- Uses a pre-trained Vision Transformer model
- Simple and easy-to-use interface

## Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Pillow

## How to Run

1. Install the required libraries:

pip install transformers torch pillow

2. Run the program:

python image_caption.py

3. Enter the image path when prompted to generate a caption.

This project demonstrates the practical application of Artificial Intelligence and Computer Vision for automatic image caption generation.
